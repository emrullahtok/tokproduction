HTML nedir?

HTML (HyperText Markup Language) , web sayfalarının yapı taşlarından biridir.
HTML, web sayfalarının içeriğini ve yapısını tanımlamak için kullanılır. HTML etiketleri (tags) kullanarak, metin, resim, bağlantı ve diğer medya öğelerini organize eder.

< !DOCTYPE html>

< html>

< head>

< title>Örnek HTML Sayfası

< /head>

< body>

< h1>Merhaba Dünya!

< p>Bu, ilk HTML belgemiz.

< /body>

< /html>

Bu kod parçası, bir HTML sayfasının temel yapısını gösterir:
- < !DOCTYPE html>: HTML5 belgesi olduğunu belirtir.
- < html>: Tüm HTML içeriğini kapsar.
- < head>: Sayfanın başlık ve meta bilgilerini içerir.
- < title>: Tarayıcı sekmesinde görünen sayfa başlığıdır.
- < body>: Sayfanın görünür içeriğini barındırır.
- < h1>: Başlık etiketidir ve en büyük başlık boyutunu temsil eder.
- < p>: Paragraf etiketidir ve metin bloklarını oluşturur.


Bu temel bilgilerin üzerine birçok başka özellik ve etiket ekleyebilirsin. HTML, CSS ve JavaScript ile birlikte kullanıldığında modern ve dinamik web sayfaları yaratmak için güçlü bir kombinasyon oluşturur.
